# SaaS Unit Economics Simulator 📊

An interactive unit economics simulator built with Python and Streamlit to model business growth and profitability.

## Key Features:
* **LTV & CAC Analysis:** Real-time calculation of Customer Lifetime Value and Acquisition Cost.
* **Profit Forecasting:** Visualization of cumulative net profit over a 24-month period.
* **Break-even Point:** Automatically identifies the month when the project starts generating positive cash flow.

## Model Performance Highlights:
* **Average LTV:** $480
* **Projected Net Profit (24 mo):** $3,055,152
* **Time to Break-even:** Month 2

## Tech Stack:
* **Language:** Python (Pandas, Plotly)
* **Interface:** Streamlit
* **Deployment:** Streamlit Cloud

## How to Run:
This project is configured for easy deployment on Streamlit Cloud. The main entry point is `src/app.py`.
